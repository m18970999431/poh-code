# 二、部署运行
## Development server
- 启动服务：ng serve
- 测试地址：http://localhost:4200/
 
## Build
- 普通打包：ng build
- 优化打包：ng build --prod

## Running unit tests
- 单元测试：ng test
- 测试参考：[Karma](https://karma-runner.github.io)

## Running end-to-end tests
- 点点测试：ng e2e
- 测试参考：[Protractor](http://www.protractortest.org/).

## Further help
- 更多帮助：ng help
